# ポケカ戦績ノート

ポケモンカードゲームの大会結果を、ブラウザ内に保存して分析するPWAです。

## できること

- シティリーグ／CL／ジムバトル／自主大会の記録
- 使用デッキ、順位、ラウンドごとの勝敗・先攻後攻・残りサイド・感想の保存
- 使用デッキ別、相手デッキ別、先攻後攻別の勝率分析
- JSONバックアップ／復元

## 使い方

1. `index.html` をHTTPS対応の静的ホスティングへ配置します。
2. 公開URLをスマホまたはPCで開き、「大会を記録」から入力します。
3. 必要に応じて設定画面からJSONバックアップを書き出します。

データはブラウザごと・端末ごとにIndexedDBへ保存されます。別端末への移行は、設定画面の「JSONを書き出す」「JSONを読み込む」を使ってください。

> PWAのオフライン利用・ホーム画面追加にはHTTPSでの公開が必要です（`localhost` は開発時の例外です）。
